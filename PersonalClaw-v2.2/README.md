# 🦞 PersonalClaw v2.2 - AI Digital Employee

**Your Personal AI Assistant with NO CODE EDITING REQUIRED!**

---

## ✨ What is PersonalClaw?

PersonalClaw is an AI-powered digital employee that can:
- 🤖 Chat with you using Claude or GPT-4
- 🌐 Browse the web and search for information
- 📧 Send emails and manage calendar
- 💬 Integrate with Telegram
- 🗣️ Use voice commands (speech-to-text and text-to-speech)
- 🤖 Automate repetitive tasks
- 🔒 Keep you safe with built-in Financial Blocker

---

## 🎯 Easy Installation (3 Steps!)

### 1️⃣ Install Dependencies
```bash
npm install
```

### 2️⃣ Add Your API Keys
**Option A: Using Settings GUI (Recommended)**
- Run `npm start`
- Click "Settings" button
- Add your API keys in the user-friendly interface
- Click "Save"

**Option B: Create `.env` file manually**
- See `INSTALLATION_GUIDE.md` for template

### 3️⃣ Start PersonalClaw
```bash
npm start
```

**That's it!** No code editing, no complicated setup!

---

## 🔑 Required API Keys

You need at least ONE of these:
- **Anthropic API Key** - Get from [console.anthropic.com](https://console.anthropic.com/)
- **OpenAI API Key** - Get from [platform.openai.com/api-keys](https://platform.openai.com/api-keys)

Optional integrations:
- Tavily (web search)
- Telegram (messaging)
- ElevenLabs (voice)

**All keys are added through the Settings GUI - no file editing!**

## ✅ Features

- ✅ **User-Friendly Settings GUI** - Add API keys without editing code
- ✅ **Multiple AI Providers** - Choose between Claude and GPT-4
- ✅ **Web Browsing** - Automated browser control with Playwright
