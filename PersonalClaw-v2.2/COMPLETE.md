## 🚀 How to Use

### 1. Install Dependencies
```bash
cd /tmp/PersonalClaw
npm install
```

### 2. Configure
```bash
cp .env.example .env
# Edit .env with your API keys
```

### 3. Start Everything
```bash
# Terminal 1: Gateway
npm run gateway

# Terminal 2: Main App
npm start

# Terminal 3: Desktop App
npm run desktop
```

### 4. Test All Features

**Gateway:**
```bash
curl http://localhost:18789/health
curl http://localhost:18789/api/status
```

**Agent Execution:**
```bash
curl -X POST http://localhost:18789/api/agent/execute \
  -H "Content-Type: application/json" \
  -d '{"message": "Take a screenshot"}'
```

**WebSocket:**
```javascript
const ws = new WebSocket('ws://localhost:18789');
ws.send(JSON.stringify({
  type: 'chat',
  data: { message: 'Hello!' }
}));
```

**Voice:**
- Record audio file
- Send to voice interface
- Get transcription + response

**Desktop:**
- Launch with `npm run desktop`
- Use system tray
- Try global shortcuts
- Test overlay mode

## 💡 Usage Examples

### Example 1: Browser Automation
```javascript
import browser from './src/tools/browser.js';

// Navigate (with financial blocking)
await browser.navigate('https://google.com');

// Fill form
await browser.fillForm('#search', 'AI news');

// Click button
await browser.click('button[type="submit"]');

// Extract data
const data = await browser.extractData({
  title: 'h1',
  description: '.description'
});
```

### Example 2: Voice Command
```javascript
import voiceInterface from './src/voice/voice-interface.js';

// Process voice command
const result = await voiceInterface.processVoiceCommand('./audio.wav');
// Returns: { transcription, response, audioPath }
```

### Example 3: Agent Execution
```javascript
import agentExecutor from './src/agent/executor.js';

// Execute command
const result = await agentExecutor.execute('Take a screenshot and analyze it');
// Agent will use screen watcher + LLM
```

### Example 4: Memory Storage
```javascript
import database from './src/memory/database.js';

// Save conversation
database.saveConversation('user123', 'Hello', 'user');

// Create task
database.createTask('user123', 'Review code', 'Check PR #123', 5);

// Get history
const history = database.getConversationHistory('user123', 50);
```
