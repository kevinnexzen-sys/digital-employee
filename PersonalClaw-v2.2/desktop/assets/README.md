# Desktop Assets

Place your icon files here:

- `icon.png` - Main application icon (512x512 recommended)
- `tray-icon.png` - System tray icon (32x32 recommended)

For now, the app will work without these files (Electron will use default icons).

To add custom icons:
1. Create or download PNG icons
2. Place them in this directory
3. Restart the desktop app
