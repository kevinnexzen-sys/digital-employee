# 🤖 KevinJr Installation Guide

### Prerequisites
* **Node.js** (v18 or higher)
* **OpenAI API Key** (for the AI brain)

### Setup Instructions
1. **Clone the Repo:** `git clone https://github.com/kevinnexzen-sys/digital-employee.git`
2. **Install Dependencies:** Run `npm install` in the terminal.
3. **Configure Environment:** - Rename `.env.example` to `.env`.
   - Add your API keys.
4. **Launch:** Run `npm start` to bring KevinJr online.
